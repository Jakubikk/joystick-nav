A one-page reference for using the Meta Quest 3 DecartAI app.

```

│  LEFT CONTROLLER    RIGHT CONTROLLER    │
├─────────────────────────────────────────┤
│  [Trigger]          [Trigger]           │
│  ⬅️ GO BACK         ➡️ CONFIRM          │
│                                         │
```

## ⌨️ Custom Prompt Quick Guide

**How to Use:**
1. Press **A Button** to open keyboard
2. Type your transformation
3. Right Trigger to submit

**Example Prompts:**
- "Transform into a pirate ship"
- "Make everything look like candy"
- "Medieval dragon's lair"
- "Underwater palace with mermaids"
- "1960s psychedelic poster"
- "Steampunk laboratory"

**Tips:**
- Be specific and descriptive
- Use 15-30 words
- Include visual details
- Mention colors, materials, atmosphere

## ⚙️ Settings & Performance

**Internet Required:**
- Minimum: 8 Mbps
- Recommended: 20+ Mbps
- Use 5GHz WiFi

**Latency:**
- First transformation: 5-10 sec
- After: 3-5 sec
- Real-time feel: ~150-200ms

**Performance Tips:**
- Close other apps
- Strong WiFi signal
- Let Quest cool if hot
- Restart app if issues

## 🐛 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| Menu not showing | Press Start (☰) button |
| Controls not working | Check controller batteries |
| No transformation | Check WiFi connection |
| Keyboard won't open | Press A button in Custom Prompt |
| App laggy | Close background apps, check WiFi |
| Camera not working | Grant permissions in Settings |

## 💡 Pro Tips

1. **Best Results:**
   - Strong WiFi (5GHz)
   - Good lighting
   - Stand ~6 feet from camera

2. **Battery Life:**
   - ~2 hours continuous use
   - Close app when not in use
   - Monitor Quest battery level

3. **Content Creation:**
   - Record using Quest's built-in recorder
   - Screenshot with Quest button combo
   - Share on social media

4. **Navigation Speed:**
   - Navigate slowly for better control
   - Wait for transformations to complete
   - Don't spam buttons

## 🆘 Emergency Fixes

**App Frozen:**
1. Press Oculus button
2. Quit app
3. Relaunch

**Lost in Menu:**
1. Press Left Trigger to go back
2. Press Start to hide/show menu
3. Restart app if stuck

**No Response:**
1. Check controller tracking
2. Re-pair controllers
3. Restart Quest headset
