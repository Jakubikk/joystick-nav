# DecartXR Quick Reference Card

## 🎮 Controller Mapping

| Button/Input | Action |
|--------------|--------|
| **Left Joystick ↕** | Navigate menu up/down |
| **Left Joystick ↔** | Adjust time slider (Time Travel mode) |
| **Right Trigger** | Confirm/Select option |
| **Left Trigger** | Go back to previous menu |
| **Start Button ≡** | Show/Hide menu |

> **Note**: No other buttons are used. Everything is controlled with joystick and triggers!

---

## 📋 Features Quick Access

### 🕰️ Time Travel
- **What**: View environment in different historical eras
- **Range**: Year 1000 to 3000
- **Controls**: Joystick left/right to adjust year, Right trigger to apply
- **Tip**: Wait 3-5 seconds after selecting year

### 👔 Virtual Dressing Room
- **What**: Try on 20+ different outfits
- **Options**: Business, casual, fantasy, historical, occupational
- **Controls**: Joystick up/down to browse, Right trigger to apply
- **Tip**: Face the camera directly for best results

### 🌍 Biome/Country
- **What**: Transform to 20+ different locations
- **Options**: Natural biomes, famous cities, cultural locations
- **Controls**: Joystick up/down to browse, Right trigger to apply
- **Tip**: Try both natural and urban locations

### 🎮 Video Game Styles
- **What**: See world in 25+ game aesthetics
- **Options**: Minecraft, Zelda, GTA, Cyberpunk, and more
- **Controls**: Joystick up/down to browse, Right trigger to apply
- **Tip**: Different games have very different visual styles

### ⌨️ Custom Input
- **What**: Type your own creative prompts
- **Access**: Right trigger opens keyboard
- **Submit**: Press Enter or A button
- **Tip**: Be specific and descriptive for best results

## 💡 Pro Tips

### For Best Results
- ✅ Ensure good lighting
- ✅ Stand 3-6 feet from walls
- ✅ Wait 3-5 seconds after selection
- ✅ Use stable WiFi (8+ Mbps)
- ✅ Face camera for clothing try-on

### Navigation Tips
- **Lost in menus?** Press Left Trigger to go back
- **Menu in the way?** Press Start button to hide it
- **Want to explore?** Use Custom Input for unlimited options
- **Not working?** Wait a bit longer, AI processing takes time

### Custom Prompt Ideas
- "Transform into a magical fairy tale forest"
- "Make everything look like watercolor painting"
- "Turn this into ancient Rome at sunset"
- "Convert to Studio Ghibli animation style"
- "Make it look like I'm on Mars"

## 🔧 Troubleshooting

| Problem | Solution |
|---------|----------|
| **Black camera** | Grant camera permissions, check lighting |
| **No AI processing** | Check internet (8+ Mbps), wait longer |
| **Menu not responding** | Check controller batteries, restart app |
| **Wrong transformation** | Be more specific in prompts, wait longer |
| **App crashes** | Restart Quest, clear cache, reinstall |
